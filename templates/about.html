About the Project

This project is a Flask web application that provides autocorrection suggestions based on a predefined dictionary of words.

The autocorrection algorithm calculates the similarity between the input word and the words in the dictionary using the Jaccard distance, and suggests the most similar words.

The application uses Flask, a Python web framework, to handle HTTP requests and render HTML templates.

Feel free to try it out by entering a word in the search box on the home page!
